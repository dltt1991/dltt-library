## 一、赛道判断：为什么是现在？

### 1.1 政策定调：算电协同从"行业探索"升级为"国家战略"

- **2026年3月**，《政府工作报告》首次将"算电协同"写入"超大规模智算集群、算电协同等新基建工程"，标志其从企业自发探索上升为国家战略部署。
- **2026年5月**，国家发改委、能源局、工信部、数据局四部门联合印发《促进人工智能与能源双向赋能的行动方案》，专章部署算力电力高效协同，明确价格信号引导算力跨区调度、挖掘算力柔性调节潜力。
- **"十五五"规划纲要**要求"推动绿色电力与算力协同布局"，国家能源局提出"三步走"推进路径，绿电直连"点对多"共享模式已有99个项目获批、对应装机3405万千瓦。

### 1.2 市场爆发：算力正在吃掉电力系统的增量

- **需求端**：2026年Q1互联网数据服务业用电量229亿千瓦时，同比增44%；2025年算力中心用电量达1960亿千瓦时，预计2030年超7000亿千瓦时，占全社会用电量5%以上。
- **成本端**：电力占数据中心运营成本的56.7%~70%，其中电费（IT负载）占35%，冷却系统占30%，电力相关合计占OPEX的65%。
- **市场规模**：2026年中国算电协同市场规模约1800亿元（增速超50%），预计2030年突破1万亿元；截至2026年2月，全国已建成算力协同项目超120个，在建超80个，总投资超5000亿元。

### 1.3 核心矛盾：掌握AI成本主动权，是算电协同的终极博弈

AI产业正在从"大力出奇迹"的粗放扩张，进入**"成本定生死"的精细化运营阶段**。谁掌握AI成本主动权，谁就掌握下一代AI基础设施的话语权。

**成本主动权有两个战场，缺一不可：**

**战场一：硬件成本。** 英伟达GPU垄断导致算力硬件成本居高不下，国产芯片（昇腾、寒武纪、摩尔线程）性价比突出但生态薄弱，"能不能用、好不好用"成为关键瓶颈。**通过AI Infra软件层的异构部署与推理加速，让国产高性价比芯片跑出国际主流性能，是从软件层重新定义硬件成本结构的唯一路径。**

**战场二：能源成本。** AI算力成本中电费占60%~70%，电力是算力最大的可变成本。中国模型推理报价仅为海外1/6~1/10，但电力成本仍是压缩空间最大的环节。**通过算电协同直连低成本绿电，同时用AI反哺电厂帮其增收降本，从而锁定更优惠电价，是从能源层掌握成本主动权的唯一路径。**

**两个战场叠加，形成"双降本"效应**：软件层降低硬件门槛，能源层降低电力成本，两者共同指向一个结果——**让每一度电产出最多的Token，让每一个Token的成本做到全球最低。**

> **本质逻辑**：电力难以跨境，但Token让"电留在国内、价值流向海外"——1度0.5元的绿电，经GPU集群+大模型推理转化为约550万Token，商业价值可达10元/度，**增值20倍**。而掌握成本主动权者，能把这20倍增值中的大部分转化为利润壁垒。

> **对投资人的价值翻译**：当行业还在卷"谁能买到更多英伟达H100"时，我们用软件能力让客户**"用国产芯片、付更低价格、跑同等性能"**。这是从根上掌握AI硬件成本主动权。

## 三、能力验证：从自动驾驶到电力预测

### 3.1 自动驾驶：端到端预测任务的技术底座

- **端到端轨迹预测**：基于BEV+Transformer架构，实现他车轨迹预测（Motion/OccFormer）与自车轨迹规划（Planner）的端到端闭环。
- **实车验证**：大弯道巡航、汇出主路、变道避让、路口连续选道等复杂场景，高速智驾百公里安全接管<0.3次，自动泊车泊入成功率>98%。
- **工程能力**：50TOPS中低算力平台做出有竞争力的智驾体验，推理算力节省30%，全场景无图通行。

### 3.2 电力预测：从"轨迹预测"到"价格预测"的技术迁移

- **六大因子体系**：基础信息（气象、日期）、系统调节安全（备用容量）、供给侧（煤风光水储）、节点价格技术因子、电网拓扑、特殊事件。
- **算法框架**：基础数据清洗→人工因子构建（学术论文+遗传规划+时序模型）→多因子组合（筛选/去相关/深度学习合并）→深度强化学习策略生成→回测评估反馈优化。

### 3.3 算法工程化：让AI在极端算力约束下跑出商业价值

- **端侧大模型**：行业首发内座舱端侧多模态大模型，基于高通平台实现9项多模能力建设，多模解码速率约30toks/s。
- **推理优化**：KV Cache、量化压缩、混合设备推理（CPU+GPU+NPU）、引擎流水调度。

## 四、我们要做什么？构建"算电双向飞轮"，掌握AI成本主动权

### 4.1 核心定位：算电协同场景下的"AI成本主动权掌握者"

**我们不建算力中心，不持重资产，不做硬件集成。**

我们的角色是**AI Infra软件枢纽**——但不同于单纯的工具厂商，我们是**"算电双向飞轮"的软件定义者**，通过两个维度掌握AI成本主动权：

**维度一：软件层掌握硬件成本主动权。** 通过异构部署+推理加速，让客户摆脱英伟达单一依赖，用国产高性价比芯片跑出同等性能，从根上降低算力硬件成本。

**维度二：能源层掌握电力成本主动权。** 通过算电协同直连电厂绿电，同时用AI赋能电厂帮其增收降本，从而锁定长期低价电力协议，从根上降低算力能源成本。

### 4.2 核心解法：轻资产模式下的"双降本"飞轮

当前算力产业链三类玩家各自存在成本断点：

| 玩家类型 | 代表企业 | 核心能力 | 成本断点 |
|----------|----------|----------|----------|
| **推理服务平台** | 硅基流动 | 模型推理加速、MaaS | 租赁标准数据中心，电力成本0.6-1.0元/度且随行就市，不可控；硬件绑定英伟达 |
| **异构算力运营商** | 无问芯穹 | 异构混训混推、算力池化 | 不持有算力资产，更不掌控能源，是"算力二房东" |
| **算力集群集成商** | 基流科技 | 万卡集群设计、硬件集成 | 硬件采购成本占80%+，电力成本转嫁给客户，自身不掌握低成本能源 |

**我们的"双降本"飞轮解法**：

**降本一：软件层打破硬件垄断。** 通过异构部署引擎，让国产高性价比芯片（昇腾、寒武纪、摩尔线程）跑通主流大模型，客户无需为英伟达支付溢价。在推理加速工具链的加持下，国产芯片的Token产出效率可达到国际主流GPU的80%-95%，但硬件采购成本仅为30%-50%。

**降本二：能源层锁定低价绿电。** 通过算电协同直连电厂，获得0.2-0.5元/度的长期绿电协议，同时用AI赋能电厂帮其增收降本，形成"我帮你赚钱，你给我低价电"的双赢契约。

**两者叠加的复合效应**：
```
传统模式成本结构：
  硬件：英伟达GPU（100%基准）
  电力：市场价0.8元/度（100%基准）
  → 单Token成本 = 高硬件成本 + 高电力成本

我们的双降本模式：
  硬件：国产芯片（30-50%成本）+ 异构部署软件优化（效率接近）
  电力：绿电长协（25-60%成本）+ AI赋能电厂（双赢锁定）
  → 单Token成本 = 行业最低水平，掌握绝对定价权
```

**飞轮闭环（掌握成本主动权的自增强机制）**：
1. 软件层降本吸引更多客户使用合作算力集群；
2. 算力收入持续投入AI Infra软件研发，异构优化能力更强；
3. 更好的AI模型反哺电厂（飞轮B），电厂收益提升；
4. 电厂愿意签订更长期、更低价、更大额度的绿电协议；
5. 更低的电力成本让我们的Token服务在全球市场更具价格竞争力；
6. 价格优势吸引更多客户，数据和收入进一步放大；
7. 继续投入研发，飞轮越转越快，成本壁垒越筑越高。

### 4.3 产品矩阵：三层软件架构，驱动"双降本"飞轮

#### 第一层：绿电Token云（MaaS）——成本优势的价值出口

**产品形态**：开箱即用的大模型API服务，覆盖语言、语音、图片、视频等场景，按Token量计费。

**运营模式**：我们不自建算力集群，而是**与算力中心建设者及电厂合作**，将他们的裸算力通过我们的AI Infra软件转化为标准化Token服务。

**"双降本"带来的定价权**：

| 维度 | 硅基流动 | 无问芯穹 | 基流科技 | 我们 |
|------|----------|----------|----------|------|
| **硬件成本** | 英伟达市场价 | 多芯片但无成本优化 | 英伟达/客户指定 | **国产芯片+软件平替，硬件成本降50-70%** |
| **电力成本** | 市场价0.6-1.0元/度 | 智算中心分成价 | 客户承担 | **绿电长协0.2-0.5元/度，电力成本降40-70%** |
| **单Token成本** | 行业基准 | 行业基准 | 行业基准 | **双降本叠加，成本可以做到行业最低** |
| **碳标签** | 无 | 无 | 无 | **绿电直连，Token自带零碳标签** |
| **飞轮能力** | 单向 | 单向 | 单向 | **双向：算力↔能源，成本自我强化** |

**核心卖点**：
- **"双降本"带来的极致成本优势**：国产芯片+绿电长协，单Token成本行业最低
- **"一度绿电 = 550万Token = 10元商业价值"**的成本故事
- 支持DeepSeek、Qwen、Llama、Stable Diffusion等主流开源模型
- 面向海外开发者的"零碳Token"差异化标签

#### 第二层：AI Infra软件平台（核心产品）——掌握硬件成本主动权的引擎

**产品形态**：面向算力中心建设者和电厂的**私有化软件授权/托管服务**，解决"建好算力中心后，如何让大模型高效跑起来，且不用绑定英伟达"的问题。

**这不是硬件，不是集群建设，而是纯软件能力输出。**

**两大核心模块**：

**模块A：异构算力部署引擎——打破硬件垄断的软件层**
- **多芯片适配**：一套软件层同时支持NVIDIA、华为昇腾、寒武纪、摩尔线程等异构芯片的大模型部署
- **模型一键迁移**：基于GraphIR中间协议，实现模型从训练框架到不同推理芯片的自动转换与优化
- **算力池化调度**：将分散的异构算力虚拟化为统一资源池，按需分配给不同推理任务
- **核心价值**：让客户**"用国产芯片的价格，跑国际主流的性能"**，从软件层掌握硬件成本主动权

**模块B：推理加速工具链——让便宜芯片跑出高效率**
- **极致量化**：PTQ/QAT量化工具，支持INT8/INT4精度，模型体积压缩75%+，推理速度提升3-10倍
- **KV Cache优化**：基于团队端侧大模型经验，针对长文本场景优化显存占用与解码速度
- **动态批处理（Dynamic Batching）**：提升GPU利用率，降低单Token推理成本
- **引擎流水调度**：CPU/GPU/NPU混合流水线，任务级负载均衡

**交付模式**：
- **软件授权**：一次性授权费 + 年度维护费（纯软件毛利率70%+）
- **托管运营**：我们派驻团队托管运营合作算力集群，按算力产出收入分成
- **SaaS化**：面向中小型算力中心提供云端异构调度SaaS

**目标客户**：
- 基流科技这类集群建设者（他们需要上层AI Infra软件让集群产生商业价值，且需要异构能力降低硬件成本）
- 电厂自建算力中心（他们有电有地，缺AI软件能力，更需要摆脱英伟达单一依赖）

### 5.2 商业闭环的财务逻辑："双降本"如何量化？

对比基流科技的重资产困境（硬件采购占80%+，毛利率21.8%，现金流差），我们的**轻资产"双降本"飞轮模式**从根本上重构了盈利模型：

| 对比项 | 基流科技（重资产集成） | 我们的轻资产"双降本"飞轮 |
|--------|----------------------|-------------------------|
| **资产投入** | 自建集群，硬件采购占80%+ | **不采购硬件，纯软件+运营服务** |
| **硬件成本** | 英伟达GPU，高价且受限 | **国产芯片+异构软件，硬件成本降50-70%** |
| **电力成本** | 转嫁给客户或随行就市 | **绿电长协0.2-0.5元/度，电力成本降40-70%** |
| **单Token成本** | 行业基准 | **双降本叠加，行业最低，掌握定价权** |
| **毛利率** | 项目集成16.8%，运维服务47-56% | **软件授权70%+，SaaS 70%+，Token分成40-50%** |
| **现金流** | 硬件垫资，回款周期长 | **软件预付费/年费，现金流健康** |
| **扩展性** | 建一个集群投一笔重资 | **软件复制零边际成本，快速扩展** |
| **飞轮效应** | 无，单次项目制 | **有，双降本自我强化，壁垒越筑越高** |

**"双降本"的数学模型（一度电的价值链）**：
```
传统模式：
  硬件成本：英伟达GPU集群，假设单位算力成本 = 100
  电力成本：市场价0.8元/度，假设单位Token电力成本 = 100
  → 单Token综合成本 = 200（基准）

我们的双降本模式：
  硬件成本：国产芯片（成本40）+ 异构部署软件（效率优化至90%）
            → 等效单位算力成本 = 40 / 0.9 ≈ 44（降56%）
  电力成本：绿电长协0.3元/度（成本37.5）
            → 单位Token电力成本 = 37.5（降62.5%）
  → 单Token综合成本 = 44 + 37.5 = 81.5（降59%）

即使定价仅为行业平均的70%，仍有充足利润空间。
```

**反向飞轮（双赢契约的财务逻辑）**：
```
我们提供：电力交易大脑 + 出力预测 + 预测性维护
        ↓
电厂获得：报价优化利润 + 新能源预测收益 + 设备维护降本
        ↓
电厂愿意：签订更长期低价绿电协议（从0.5元降至0.3元/度）
        ↓
我们获得：更优电力成本 → 单Token成本进一步降低 → 更强价格竞争力
        ↓
更多客户 → 更多收入 → 投入更好的能源AI研发 → 电厂服务效果更好
        ↓
飞轮加速，成本主动权越握越紧
```
